Client Authentication

The only supported authentication type is private_key_paseto (private_key_jwt that uses PASETO instead of JWT)
This type requires the following parameters to be in request:

  • client_id - The unique client identifier
  • client_assertion_type - The assertion type, it must be equal to urn:ietf:params:oauth:client-assertion-type:paseto-bearer
  • client_assertion - The v4 public PASETO token with claims required for client assertion, signed by pre-registered client's key
