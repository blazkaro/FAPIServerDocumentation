The required parameter for every request is grant_type. Also, every request to this endpoint must be authenticated, see Client Authentication
Also, for every request you can use DPoP header. Normally it's optional, but sometimes required. See Demonstrating Proof Of Possession

Authorization Code grant

Required parameters

  • code
  • code_verifier
  • redirect_uri

Client Credentials Grant

Required parameters

  • authorization_details - see Rich Authorization Requests. The openid type will be ignored while using this grant

Refresh Token Grant

Required parameters

  • refresh_token

Optional parameters

  • authorization_details_types - if you don't want to get all earlier granted authorization_details, you can specify space-delimited types you want now, e.g openid photo-api
  • claims - if you don't want to get all earlier granted claims, you can specify space-delimited claims you want now, e.g email

Client Initiated Backchannel Authentication

Required parameters

  • auth_req_id
